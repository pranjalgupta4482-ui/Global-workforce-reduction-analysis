# 🌍 Global Workforce Reduction Analysis (2020–2026)

## 📊 Project Overview

This project analyzes global workforce reduction trends from 2020 to 2026, uncovering key insights across industries, companies, and geographies.

The goal is to identify patterns in layoffs, understand driving factors, and visualize trends using interactive dashboards.

---

## 🛠️ Tools & Technologies

* Excel (Data Cleaning & Preparation)
* Python (Pandas, Data Analysis)
* Tableau (Dashboard & Visualization)

---

## 📂 Project Structure

* **data/** → Raw and cleaned datasets
* **python/** → Data cleaning and analysis notebooks
* **tableau/** → Tableau dashboard file
* **images/** → Dashboard screenshots

---

## 📈 Key Insights

* Layoffs peaked in 2023, reflecting post-pandemic workforce corrections
* E-commerce and IT Services sectors contributed the highest layoffs
* Workforce reductions are concentrated in major tech hubs like Seattle and Bangalore
* A small group of companies drives the majority of layoffs (Pareto pattern)

---

## 📊 Dashboard Highlights

### 🔹 Overview Dashboard

* KPI cards (Total layoffs, Companies affected, Impact %)
* Yearly and monthly trend analysis
* Pareto analysis by company
* Global map visualization

### 🔹 Deep Analysis Dashboard

* Company-level workforce reduction heatmap
* City-level distribution of layoffs
* Volume vs impact analysis
